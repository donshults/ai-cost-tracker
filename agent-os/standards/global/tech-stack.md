# Technology Stack

## Backend
- **Language**: Python 3.12
- **Framework**: FastAPI
- **ORM**: SQLAlchemy 2.0 (async)
- **Migrations**: Alembic
- **Validation**: Pydantic v2

## Database
- **Primary**: PostgreSQL (Neon)
- **Connection**: asyncpg

## External APIs
- **Anthropic**: Admin/Usage API for cost data
- **Context Vault**: Summary storage

## Development Tools
- **Formatting**: black, isort
- **Linting**: flake8, mypy
- **Testing**: pytest, pytest-asyncio

## Deployment
- **Platform**: Railway
- **Container**: Docker
